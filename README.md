# О проекте

Этот проект представляет собой дашборд, созданный с использованием Tableau, для анализа данных о вакансиях аналитиков, опубликованных на сайте [Kaggle](https://www.kaggle.com/datasets/lludop/data-nalyst-vacancies-on-headhunter-for-2023-2024?resource=download).

## Цели анализа

- **Динамика количества вакансий на аналитиков по месяцам** – показывает изменения спроса на специалистов.
- **Средняя зарплата аналитиков по типу в России** – анализирует различия в доходах между различными типами аналитиков (например, Junior, Middle, Senior).
- **Топ-10 городов по количеству вакансий** – выделяет города с наибольшим количеством предложений работы.

Кроме того, в дашборде реализованы **динамические фактоиды**, которые отображают ключевые показатели проекта (например, максимальное значение зарплат или количество вакансий за определенный месяц).

## Структура проекта

1. **Источник данных**
   - Данные взяты с Kaggle. Набор содержит информацию о вакансиях аналитиков на платформе HeadHunter за 2023-2024 годы.

2. **Ключевые элементы дашборда**
   - Area chart динамики количества вакансий в разрезе месяцев.
   - Pie chart среднего уровня заработной платы по типу аналитиков.
   - Bar chart, демонстрирующий распределение вакансий по регионам, с выделением топ-10 городов.

3. **Особенности проекта**
   - Интерфейс с удобной навигацией.
   - Динамические элементы, которые обновляются в зависимости от выбранных фильтров.
